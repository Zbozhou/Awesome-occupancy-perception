# Awesome-occupancy-perception

This repository is a paper digest of recent advances in occupancy perception. 

本仓库由[公众号【自动驾驶之心】](https://mp.weixin.qq.com/s/EfrWi-yXRGowOIkCDEsmfw)团队整理，欢迎关注，一览最前沿的技术分享！
